Research Interests

I am interested in the emerging area of code provenance, which aims to provide sufficient historical context for a code base to help developers understand and maintain it better. The two current repositories that I am actively maintaining are codeHistories and webActivities. I am also broadly interested in applications of machine learning and how we can potentially transform the current programming development environment to be more change-aware and user-understanding.
