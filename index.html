Greetings! My name is Tri (Tree). I am currently a fourth year CSE PhD student at Washington University in St. Louis, fortunate to be advised by Dr. Caitlin Kelleher. My current work focuses on designing and developing tools to help end-users archive their programming data with more contexts and insights for better comprehension and learning. We have developed a system framework that can automatically capture the code stages and corresponding web browsing activities of users while they are working on programming tasks. For more information, you can check out the paper here. Currently this system has been updated to provide a more user-friendly interface and is being used as part of my current research and thesis topic on context recovery following interruptions. For any interest, feel free to reach out to me at p[dot]tri[at]wustl[dot]edu.
